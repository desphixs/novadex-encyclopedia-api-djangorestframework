# NovaDex: A Sci-Fi Creature Encyclopedia API

### Project Description
You are going to build NovaDex, a database API for a sci-fi creature encyclopedia. Each alien species belongs to a home planet, has a name, a description, and a danger level rating. The twist is that planets and creatures are separate pieces of data that are linked together — one planet can have many creatures living on it. This is called a One-to-Many relationship, and it is one of the most common patterns in real-world databases. You will build the models to store this relationship, wire up serializers to display it correctly in JSON, and create clean API endpoints for creating, reading, updating, and deleting both planets and creatures. You will also learn pagination, which is the technique every real API uses to avoid sending thousands of records at once.
